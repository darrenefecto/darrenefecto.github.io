What's a DirectX GameEngine?

A DirectX GameEngine uses Microsoft's DirectX API to handle tasks such as creating and rendering 3D graphics, playing audio, and handling user input.
it has a bunch of tools for making it easier to create your games instead of building everything from scratch.

how to create a DirectX GameEngine?

To make your own DirectX GameEngine, you first need to understand how DirectX and its different features and functions work.
Then, you need to write code for the graphics(engine) using DirectX. After that you can add features like input handling or audio playback.

if you're done with the DirectX GameEngine?

Once you have the GameEngine set up, you can start to build your game.
At the end, you can write code for functions in the Game like creating components, collisions, physics and also terrains.
